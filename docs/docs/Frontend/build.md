# Build App Production Mode

**IOS**
```sh
npm run ios -- --mode="Release"
```

**Android**
```sh
npm run android -- --mode="Release"
```

# Build App Detox Mode

**IOS**
```sh
sh build-detox-ios.sh
```

**Android**
```sh
sh build-detox-android.sh
```
